# CINE MATCH - Movie Recommendation System

A Movie Recommendation System built using Python, NLP, and Streamlit.

## Features
- Content-based movie recommendations
- NLP-based similarity calculation
- Interactive Streamlit web application
- Movie search and recommendation

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit

## Project Structure

├── app.py
├── main.py
├── movies.ipynb
├── requirements.txt
└── README.md

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/movie_recommendation_system.git
cd movie_recommendation_system
```

2. Install Dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
streamlit run app.py
```

## Dataset

The project uses a movie dataset containing movie metadata and descriptions for generating recommendations.
